# GatherHub API Documentation

API documentation will be added here as endpoints are implemented.

## Base URL

- **Local development**: `http://localhost:3000`
- **Docker (via Nginx)**: `http://localhost:80`

## Available Endpoints

| Method | Path      | Description            |
|--------|-----------|------------------------|
| GET    | `/`       | API status             |
| GET    | `/health` | Health check + DB ping |

More endpoints will be documented here as features are built.
